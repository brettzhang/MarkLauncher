# MarkLauncher 安装指南

## 快速安装

### 方法一：开发者模式安装（推荐）

1. **下载项目文件**
   ```bash
   # 如果你有项目源码，确保所有文件都在同一个文件夹中
   ```

2. **打开Chrome扩展管理页面**
   - 在Chrome地址栏输入：`chrome://extensions/`
   - 或者通过菜单：Chrome菜单 → 更多工具 → 扩展程序

3. **启用开发者模式**
   - 点击页面右上角的"开发者模式"切换开关
   - 启用后会显示额外的按钮选项

4. **加载扩展**
   - 点击"加载已解压的扩展程序"按钮
   - 在弹出的文件选择器中，选择包含项目文件的整个文件夹
   - 确保文件夹包含 `manifest.json` 文件

5. **验证安装**
   - 安装成功后，在扩展列表中会看到"MarkLauncher"
   - 打开新标签页应该会显示MarkLauncher界面

### 方法二：打包安装（用于分发）

1. **打包扩展**
   ```bash
   zip -r marklauncher.zip . -x "*.git*" "README.md" "INSTALL.md"
   ```

2. **安装打包文件**
   - 用户下载 `marklauncher.zip`
   - 解压缩到本地文件夹
   - 按照方法一的步骤2-4进行安装

## 安装后设置

### 权限说明
安装时Chrome会提示以下权限请求：
- **访问书签**：读取和管理您的Chrome书签
- **存储数据**：保存用户设置和偏好
- **访问标签页**：创建新标签页打开书签

这些都是实现核心功能所必需的权限。

### 首次使用
1. 安装完成后，打开新标签页
2. 扩展会自动加载您的Chrome书签
3. 左侧显示一级文件夹分类
4. 右侧显示选中文件夹的书签
5. 可以使用顶部搜索框搜索书签

## 卸载方法

1. 打开Chrome扩展管理页面：`chrome://extensions/`
2. 找到"MarkLauncher"扩展
3. 点击"移除"按钮
4. 确认移除

## 常见问题

### Q: 安装后新标签页没有变化？
A:
- 确认开发者模式已启用
- 检查是否正确加载了包含 `manifest.json` 的文件夹
- 尝试禁用其他可能冲突的新标签页扩展
- 刷新新标签页（Ctrl+R）

### Q: 扩展显示"无法加载书签"错误？
A:
- 检查Chrome的书签权限是否被阻止
- 尝试重新安装扩展
- 确认Chrome版本在88以上

### Q: 某些书签的favicon图标不显示？
A: 这是正常现象，一些网站可能不提供标准的favicon图标。扩展会显示默认图标。

### Q: 搜索功能不工作？
A: 确保输入的关键词正确，搜索会匹配：
- 书签标题
- URL地址
- 文件夹名称

## 浏览器兼容性

- **Chrome**: 88+ ✅
- **Edge**: 88+ (基于Chromium) ✅
- **Firefox**: 不支持（使用不同的扩展API）
- **Safari**: 不支持（使用不同的扩展API）

## 技术支持

如果遇到问题，请：
1. 查看Chrome开发者工具的控制台错误信息
2. 确认所有必需的文件都存在
3. 检查网络连接（某些功能需要网络）
4. 重新安装扩展

## 更新扩展

当有新版本时：
1. 下载新版本文件
2. 在扩展管理页面点击"更新"按钮
3. 或者先移除旧版本，然后重新安装新版本
