## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Tab` | 在搜索框中切换搜索模式 |
| `Ctrl+K` | 聚焦搜索框 |
| `Escape` | 清除搜索 / 关闭弹窗 |
| `Enter` | 执行搜索 |

## 🔧 技术栈

- **前端**：HTML5 + CSS3 + ES6+ JavaScript
- **框架**：原生JavaScript（无第三方依赖）
- **样式**：CSS变量 + Grid + Flexbox
- **存储**：Chrome Storage API
- **API**：Chrome Bookmarks API + Tabs API
- **构建**：无构建工具，直接运行
