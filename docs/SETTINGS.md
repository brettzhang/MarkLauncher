# MarkLauncher 设置功能说明

## 功能概述

MarkLauncher v1.0.0 新增了完整的设置功能，允许用户自定义搜索引擎选择和查看应用信息。

## 设置界面

### 访问方式
- 点击顶部右侧的设置按钮（齿轮图标）
- 设置弹窗以模态框形式显示

### 界面布局

## 功能说明

### 1. 搜索设置

**支持的搜索引擎：**
- **Google**：`https://www.google.com/search?q=`
- **Bing**：`https://www.bing.com/search?q=`
- **百度**：`https://www.baidu.com/s?wd=`

**使用方式：**
1. 点击左侧导航的"搜索设置"
2. 选择偏好的搜索引擎
3. 设置会自动保存并在下次搜索时生效

**搜索模式：**
- 在搜索框输入关键词
- 切换到Google搜索模式（点击"Google"按钮或按Tab键）
- 按Enter键执行搜索，会使用设置的搜索引擎

### 2. 关于页面

**显示信息：**
- 扩展名称：MarkLauncher
- 版本号：1.0.0
- 描述：简洁高效的Chrome书签启动器扩展
- 主要功能列表

## 技术实现

### 设置存储
- 使用Chrome Storage API (`chrome.storage.sync`)
- 设置数据在不同设备间同步
- 存储键名：`marklauncher_settings`

### 数据结构
```javascript
{
    "searchEngine": "google",  // "google" | "bing" | "baidu"
}
```

### 核心方法

1. **设置管理**
   - `loadSettings()` - 加载保存的设置
   - `saveSettings()` - 保存当前设置
   - `openSettingsModal()` - 打开设置弹窗
   - `closeSettingsModal()` - 关闭设置弹窗

2. **界面交互**
   - `bindSettingsEvents()` - 绑定设置相关事件
   - `switchSettingsPanel()` - 切换设置面板

3. **搜索集成**
   - `performGoogleSearch()` - 根据设置执行搜索引擎搜索

## 样式设计

### 响应式布局
- **桌面端**：左侧导航 200px，右侧功能区域
- **移动端**：顶部横向导航，下方功能区域

### 动画效果
- 弹窗显示：缩放 + 透明度动画
- 面板切换：右侧滑入动画
- 按钮交互：hover状态和焦点状态

### 主题一致性
- 使用全局CSS变量（`--primary-color`, `--border-color`等）
- 保持与主界面的视觉一致性
- 支持深色/浅色主题适配

## 快捷键支持

设置弹窗保持与主界面一致的快捷键：
- `Escape`：关闭设置弹窗
- 设置弹窗外点击：通过遮罩层关闭

## 权限要求

在 `manifest.json` 中需要以下权限：
```json
{
    "permissions": [
        "bookmarks",
        "storage"
    ]
}
```

- `bookmarks`：访问Chrome书签数据
- `storage`：保存和读取用户设置

## 兼容性
